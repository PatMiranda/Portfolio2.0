Projects

Dungeons & Dragons

A project meant to involve pushing and pulling data from Google Firebase. Instead of creating a "Rock Paper Scissors" game, I decided to experiment with more complex sets of data stored in objects. The characters are dynamically created from Firebase based on character stats stored in an array of objects.

Eldritch Investigators

This is a combination of two early assignments. 1: Hangman, introduction to javascript. 2: Crystals, introduction to Jquery. I linked the games together through a story, and added music cues and integrated VO, to gain initial experience with audio event listeners.

KidReads

A project integrating multiple new technologies. 1. Turn.js: Creates an animated book in the DOM and utilizes it's own rules for event Listeners. 2. Swiper.js: Creates the sliding bar of images. 3. Amazon Polly: And API that passes in 'clicked' words and pronounces them outloud. This project was intended to introduce a whole new format in which to publish books for kids!
